### Introduction
This is a project that allows you to play the Liar Game using AI-generated images. <br/>
This project was created for submission to a department promotion project competition. <br/>
It was created with the idea of how middle school students could use AI in an interesting way.

### Key Features
- AI image generation feature
- Basic features of the Liar Game
  - Topic selection
  - Liar selection, etc.

## Other
- This repository is for the **web frontend**.
- The backend server repository is [art-imposter-server](https://github.com/pybsh/art-imposter-server).
